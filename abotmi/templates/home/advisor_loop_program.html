Advisor Loop Program

Terms & Conditions

The REIA Advisor Loop Program allows registered REIA users (an “REIA User”) to earn referral points by referring people to REIA.

In order to participate first the REIA users must agree to these terms, which become part of the REIA T&C. Capitalized terms that are not defined here will have the same meaning given to them in the REIA Terms & Conditions.

How to earn referral points

REIA users can earn referral points if :

  • A Referred person clicks on their referral link to create a valid REIA account that complies with REIA T&C
  • The referred person shall further completely fill the Advisor Loop Program (ALP) form either as a guest or host.

Referral Points Earned

  • Every time a REIA user refers a person , REIA user earns 10 points
  • Every point is valued at Rs 25
  • REIA user can redeem these points at the end of every year
  • On successful registration of the person, REIA user gets 10 points
  • On certification of the advisor, REIA user gets 10 points
  • On first transaction of the advisor referred, the REIA user gets 10 points
  • On Crisil or Signzy verification of the advisor who is referred REIA user gets additional 10 points
  • Every time when that advisor referred refers in any other advisor, REIA user gets 2 points
  • Every time when the other advisor referred does a transaction, REIA user gets 1 point
  • When user gets 100 points, a bonus of 5 points is given
  • On every 100 points reached by user, an additional bonus points are given
  • When the advisor referred by user, achieves 100 points, additional bonus points are given and every time when they earn 100 points, user earns 1 point for that

Sharing Referral Links

Referrals should only be used for personal and non-commercial purposes and only shared with connections that will appreciate receiving these invitations. Referral links should not be published or distributed.

Multiple Referrals

A referred person may use one referral link. If a referred friend receives referral links from multiple REIA Users, only the corresponding REIA user of the referral link used by the referred person receive referral points

Severability

If any provisions in these terms are held to be invalid, void, or unenforceable , such provisions will be struck.

Termination

REIA may suspend or terminate the Advisor Loop Program or a user’s ability to participate in the Advisor Loop Program at any time and for any reason. REIA reserves the right to suspend account or remove referral points due to activity which is considered as abusive or fraudulent.

Updated Terms

REIA reserves the right to update these terms without any prior notice. The updated terms will be posted on the website and will become effective upon posting.
